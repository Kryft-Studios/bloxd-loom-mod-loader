# bloxd-loom-mod-loader
A Mod loader for running **World Code** from **Code Blocks**

# Usage
```js
const globalThis = LOOM()
tick=()=>{
api.log("hi")
}
```

# Features
- `globalThis` isolation between mods ✅
- Define events from **Code Blocks**
